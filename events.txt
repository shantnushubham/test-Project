Event is any user interaction with the web application
We can execute any JavaScript code/function whenever an event occurs

Examples of events:
1. Clicking with mouse
2. When the page has loaded
3. When some input text has been modified
4. When the user presses a key on the keyboard
5. When we hover over a certain element on the page
